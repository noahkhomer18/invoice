# Invoice Maker

A free, open-source invoice maker for creating professional invoices without subscription fees.

## About

I've been using an app called Clockify for my invoices, however I pay for that so I understand that not everyone wants to have paid apps for their solution. That is why this open source invoice maker lets you make invoices for free. At the end simply save as PDF rather than print. I will try to keep adding more functionality but lately been busy with work a lot too.

## Features

- **Free & Open Source** - No subscriptions, no fees, completely free to use
- **Professional Invoices** - Create clean, professional invoices for any business
- **Easy to Use** - Simple form-based interface, no learning curve
- **PDF Export** - Save invoices as PDF for easy sharing and record keeping
- **Auto-calculations** - Automatic subtotal, tax, and total calculations
- **Company Info Memory** - Saves your company information for future invoices
- **Multiple Line Items** - Add as many items as needed to your invoice
- **Customizable** - Add notes, payment terms, and custom information

## Usage

1. Open `index.html` in your web browser
2. Fill in your company information (saved for future use)
3. Enter customer details
4. Add invoice number, dates, and payment terms
5. Add line items with descriptions, quantities, and prices
6. Set tax rate and discounts if needed
7. Click "Generate Invoice" to preview
8. Use "Print" to save as PDF (browser print dialog â†’ Save as PDF)

## Installation

No installation required! Just download the files and open `index.html` in any modern web browser.

## Files

- `index.html` - Main invoice form and structure
- `styles.css` - Styling and layout
- `script.js` - Invoice functionality and calculations
